# ELS Experience Update Requirements

## 1. Add a Home → ELS Experiment Navigation Button

The Home page currently does not clearly guide users to the ELS experience.

Create a large and visually prominent button on the Home page.

Example:

🎲 ELS 실험 시작하기

Subtext:

"직접 체험하며 ELS를 배워보세요!"

Requirements:

* Large card design
* Easy to notice
* Mobile friendly
* One-click navigation to the ELS Experience page
* Use large icons and large text

The ELS Experience should be one of the main actions on the Home page.

# 6. Language Requirements

All text shown on the website must be written in Korean.

Do not display English in the UI.

Use vocabulary that elementary school students can understand.

If a difficult financial term must be used, provide a tooltip explanation.

Example:

원금 손실 (?)

"처음 넣은 돈보다 적게 돌려받는 것을 말해요."

기초자산 (?)

"ELS 결과를 결정하는 기준이에요."

---

# 7. Design Goal

The page should feel like:

* 어린이 금융 체험관
* 쉬운 투자 학습 게임
* 인터랙티브 스토리

The page should NOT feel like:

* 금융 보고서
* 투자 설명서
* 긴 블로그 글

Users should learn ELS naturally through large visual cards, simple explanations, and interactive choices.
