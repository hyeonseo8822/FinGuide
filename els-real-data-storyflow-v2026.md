# ELS Experience Page Revision (Real Data Version)

## Project Goal

The purpose of this page is not to create a random simulation.

The purpose is to help students understand how ELS actually works through a real example.

Use a real Samsung Electronics case instead of randomly generated companies and stock prices.

The page should feel like an interactive learning experience rather than a financial report.

---

# 1. Add an Introduction Screen Before "My Choice"

Do not start with the choice screen.

First, show a simple comparison between ELS and Savings Deposits.

Title:

## 어떤 차이가 있을까요?

Display two visual cards.

### 🐷 예금

* 원금 보호
* 결과가 정해짐
* 위험 낮음
* 수익 적음

### 🎲 ELS

* 원금 손실 가능
* 결과가 달라질 수 있음
* 위험 높음
* 수익 높음

Requirements:

* Use large icons.
* Use very short keywords.
* No long explanations.
* Make it understandable within a few seconds.

Button:

### 직접 선택해 보기

Clicking the button moves to the next screen.

---

# 8. Language Requirements

All visible content must be written in Korean.

No English should appear in the user interface.

Use simple Korean that elementary school students can understand.

Examples:

❌ Principal Loss

✅ 원금 손실

❌ Stock Price

✅ 주가

❌ Success

✅ 성공

---

# 9. Typography and Visual Hierarchy

Make important information larger.

Especially:

* 성공
* 실패
* 최종 금액
* 원금 손실 가능
* 예금 vs ELS 비교

Use:

* Large cards
* Large icons
* Simple graphics
* Clear visual hierarchy

Avoid long paragraphs.
